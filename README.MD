# How to use the project

## MPESA Dynamic QR Code
Open ``mpesa.php`` and add your own sandbox consumer secret and consumer keys

## App
Ensure you have the requirements to run an ionic app:
    - You may need a php server ie
    - Xampp
    - Wampp
    - Online Server
